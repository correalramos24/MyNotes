# MyNotes
A simple but detail-oriented task and notes app.


## Capabilities (so far)
* Store notes, with plain text and from a file
* Store simple tasks

# USAGE

```
export PYTHONPATH="path/to/MyNotes/root/folder"
python3 app/main.py
```
